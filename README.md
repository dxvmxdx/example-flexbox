# example flexbox

flexbox를 이용한 레이아웃 예제

<br>

## 📐 flex item의 width, height 계산

flex item의 너비, 높이 값은 (지정하지 않은 경우)  
메인축과 같은 방향이면 item의 content만큼  
교차축과 같은 방향이면 container에 `align-items`값이 없다면 container 크기를 가진다.
